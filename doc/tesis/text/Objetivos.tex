\section*{Objetivo General}
\markboth{\MakeUppercase{Objetivo General}}{}
\addcontentsline{toc}{section}{Objetivo General}
Crear un software libre capaz de interpretar las señales de un EEG para que puedan ser usadas por medio de herramientas externas.

\section*{Objetivos Particulares}
\markboth{\MakeUppercase{Objetivos Particulares}}{}
\addcontentsline{toc}{section}{Objetivos Particulares}

\begin{itemize}
\item Establecer un canal de comunicación entre el software y el EEG.
\item Obtener datos numéricos a partir de las señales obtenidas.
\item Tomar esos datos en tiempo real y graficarlos en una relación frecuencia-tiempo.
\item Establecer un canal de salida para enviar los datos interpretados.
\end{itemize}
