ROS
ROS (Robot Operating System) provee librerías y herramientas para ayudar a los desarrolladores de software a crear aplicaciones para robots. ROS provee abstracción de hardware, controladores de dispositivos, librerías, herramientas de visualización, comunicación por mensajes, administración de paquetes y más[4]. ROS es utilizado en el software desarrollado para establecer una comunicación por mensajes en tiempo real.
